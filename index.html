JS Report


Components of an online system that JavaScript can be used for

  1. Microsoft builds its Edge web browser to process and execute Javascript efficiently. Microsoft has developed and maintained its own JavaScript engine for Edge, is considering creating an alternative version of NodeJS with the Edge engine, and supports Node on the Azure cloud platform.
  2. Paypal uses JavaScript in the front end of their website, and NodeJS in the back end for all client-facing applications
  3. Netflix uses JavaScript and NodeJS in the cloud to break down pieces of their user interface into individual services, alleviating stress on their servers.
  4. Groupon rebuilt their entire website be breaking down everything into individual NodeJS web applications to operate faster.
  5. Uber uses JavaScript and NodeJS to handle huge volumes of data in real time, taking advantage of its asynchronous capabilities.
  6. Facebook is dominantly based on Javascript. Each piece of a webpage is separate. It developed its own way of breaking down and delivering sections of JavaScript separately to represent the webpage as a collection of independent JavaScript application. Facebook also created the front end JS framework React.
  7. Google search uses JavaScript to display results as your typing. Gmail web client and Google docs also use JavaScript. Google docs uses Closure Tools to compile JavaScript into a lower level faster form. Google developed a JavaScript engine called V8 for its web browser Chrome, which became the basis for NodeJs. Google also created the front end JS framework Angular JS to be used in its DoubleClick ad platform.
  8. Ebay converted everything to JavaScript and NodeJS, much like Netflix
  9. Walmart uses JavaScript and NodeJS for its online retail business, and other web applications requiring multiple users to access management interfaces simultaneously
  10. Linkedin uses NodeJS for its mobile site to solve its scaling problems. It's asynchronous capabilities allow the site to perform quicker using fewer resources. It also made data sharing and building APIs easier for Linkedin developers.

Top 5 reasons to use JavaScript

  1. JavaScript is the default language of the internet and can be run and tested in a browswer without needing a special development environment.
  2. JavaScript is easy to learn compared to other languages. Syntax is simple and logical.
  3. JavaScript is highly versatile with a wide range of uses, including: web applications, mobile applications, deskop applications, big data & cloud development, blockchain and automation of day-to-day work flows.
  4. JavaScript has a large online community for support, especially StackOverflow.
  5. JavaScript opens the door to great career opportunites, as developers are in high demand.
